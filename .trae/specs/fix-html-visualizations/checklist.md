# Checklist - HTML 可视化组件优化

## 决策树
- [x] 根结点到第二层有 Y 型分叉连接线（左 Yes / 右 No）
- [x] 每层之间有垂直连接线带向下箭头
- [x] 连接线颜色与目标结点类型一致（Type-1红/Type-2橙/Type-2.PF黄/Type-3蓝/Type-4紫）
- [x] 连接线上正确标注分支标签（Yes/No/Yes(Success)/Traditional Fail/Semantic Fail 等）
- [x] 5 种结果叶结点（Type-1/Type-2/Type-2.PF/Type-3/Type-4）全部可见且连接完整
- [x] 决策树在 1280px+ 宽度下不溢出或可水平滚动
- [x] 整体视觉清晰，层级关系一目了然

## Mermaid 流程图交互
- [x] 鼠标滚轮在流程图上滚动时触发缩放（50%-200% 范围）
- [x] 缩放以鼠标位置为中心点
- [x] 鼠标拖拽可平移图表
- [x] 显示当前缩放比例指示器
- [x] 有"重置视图"按钮可恢复默认大小位置
- [x] 协作流程图（sequenceDiagram）和 L2 门控图（flowchart）均支持缩放
- [x] 缩放后图表清晰度不受影响

## 兼容性
- [x] 不影响页面其他部分（侧边导航、其他章节、图表等）
- [x] 无新增外部依赖（纯 CSS + JS 实现）
